# mako discord bot

mako is a lightweight bot designed to help with general server functionality.<br>
heres a few things mako can help you with at the moment:
## tickets
- /createticket : creates a ticket with support for adding a user<br>
- /closeticket : closes the ticket the command is ran in<br>
## moderation
- /makoKick : kicks user with a sent dm (if able)<br>
- /makoBan : bans user with a sent dm (if able)<br>
- /purge : deletes a set amount of messages from chat
## misc
- /testcommand : test to see if mako is alive!

# set-up

to be honest, its really simple youll just need a few things to get going

## 1. download the repo (duh)
if you arent sure how to do this ask someone else lol

## 2. install python
same thing, just basic terminal use

## 3. use pip to install dependencies
youll just need to run this to install everything it needs<br>
` python -m pip install discord.py python-dotenv `

## 4. set up your .env and data.json
add your api key from the discord dev dashboard in your .env change the ids for your discord categories and channels so everything works well

## 5. run the bot
just run `python index.py` where you put the bot and your good to go :)
